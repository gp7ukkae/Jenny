## 🎯 自定義指南

### 更換照片
在 `script.js` 中修改 `photoData` 陣列：

```javascript
const photoData = [
    {
        id: 1,
        src: '您的照片URL',
        title: '照片標題',
        description: '照片描述',
        category: 'friends' // friends, family, events
    },
    // 更多照片...
];
```

### 調整色彩主題
在 `styles.css` 的 `:root` 選擇器中修改 CSS 變數：

```css
:root {
    --primary-color: #你的主要色;
    --secondary-color: #你的次要色;
    /* 其他色彩變數... */
}
```

### 新增類別
1. 在 `photoData` 中新增新的 `category`
2. 在 HTML 中新增對應的篩選按鈕
3. 在 `getCategoryIcon` 和 `getCategoryName` 函數中新增對應內容
